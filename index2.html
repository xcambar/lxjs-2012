Yay, Modules!

Modules, WTF's tis shit ?

A module
  • MAY have a name
  • MUST be context-free (runnable when only its dependencies are available)
  • SHOULD have the least side-effects possible
  • MAY have dependencies
  • MAY expose an API

Modules are easy in Javascript

  • var i = {a: function () { }, BASE_VALUE: 42}
  • IIFEs define modules
  • Any function with well-defined variables returning an Object
SCEPTICAL => NOT enough for the constraints above
A module loader
  • MUST resolve dependencies
  • MUST NOT expose modules publicly
  • SHOULD NOT expose its own API
  • SHOULD have an adaptative behaviour
